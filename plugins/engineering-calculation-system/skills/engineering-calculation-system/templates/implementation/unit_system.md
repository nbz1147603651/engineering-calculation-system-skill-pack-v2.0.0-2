# Unit System

## Policy

Use one internal unit system. Convert units only at input and output boundaries.

| Quantity | Internal Unit |
| --- | --- |
| length | m |
| force | kN |
| stress_or_pressure | kPa |
| unit_weight | kN/m3 |
| moment | kNm |
| settlement_or_displacement | mm |
| angle_internal | radian |
