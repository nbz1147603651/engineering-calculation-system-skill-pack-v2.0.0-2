Use this ledger in `.engineering-calc/work/progress.md` or an equivalent project-local scratch
location. It is the recovery map after context compaction.

```text

```

Statuses: `pending`, `in_progress`, `complete`, `blocked`, `needs_review`, `rejected`.

Resume from the first task not marked `complete`. Trust the ledger and git history over chat
memory.
